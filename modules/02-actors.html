The Agents: Your Specialists

Each agent is a specialist with a specific job. Here's how they're defined:

CODE 

--- agent: planner.md ---
name: planner
description: Implementation planning
tools: [read, write, exec]
when_to_use: Complex features, refactoring
PLAIN ENGLISH

This is the planner agent's profile — like a business card.

Its name is "planner" — that's how we call it.

It handles implementation planning — breaking down big tasks.

It can read files, write new ones, and run commands.

Use it when you need to plan complex features or refactor code.

1
planner

Breaks down complex features into implementable steps — like an architect creating blueprints before construction begins.

2
tdd-guide

Writes tests first using TDD — like creating a checklist before you start packing for a trip.

3
code-reviewer

Checks code quality and maintainability — like an editor reviewing an article before publication.

4

Finds vulnerabilities and security issues — like a security consultant checking your doors and windows.

5
build-error-resolver

Fixes build and type errors — like a mechanic diagnosing why your car won't start.
